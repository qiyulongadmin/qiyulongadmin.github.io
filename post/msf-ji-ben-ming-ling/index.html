msf基本命令

2020-06-23

命令行启动msf
msfconsole

搜索命令(search)：
search cve (注：含有excellent的模块可以直接用)

进入模块(use):
use exploit/windows/smb/ms09_050_smb2_negotiate_func_index

检测(check):
check

返回上一级(back):
back

查看msf信息(banner):
banner

链接一个主机(connect):
connect www.baidu.com

显示一个或多个模块的信息(info):
info exploit/windows/smb/ms09_050_smb2_negotiate_func_index

进入irb脚本(irb):
irb

显示和管理作业(jobs):
jobs
(备注：和Windows任务管理器的进程作用一样)

杀死一个进程(kill):
kill
(和结束进程的作用一样)

加载一个模块的路径(loadpath):
loadpath exploit/windows/smb/ms09_050_smb2_negotiate_func_index

退出msf(quit):
quit

加载一个插件(load):
load

运行存储一个文件中的命令(resource):
resource

查看一个会话的路由信息(route):
route

保存动作(save):
save

给一个变量赋值(set):
set

显示所有类型的模块(show):
show options

把一个赋值给全局变量(setg):
setg
(备注：例如上述 set设置的IP,就会用到他攻击模块的RHOST中)

在限定的秒数内什么也不做(sleep):
sleep

卸载一个模块(unload):
unload

解出一个或多个变量(unset):
unset

解出一个或多个全局变量(unsetg):
unsetg

显示msf和控制台库版本(version):
version

监听,若有漏洞返回一个shell(exploit):
exploit
