vs code链接ftp

2020-07-02

1.安装插件sftp
2.ctrl+shift+p调出窗口输入sftp:config
3.配置基本信息，代码如下
{
"name":"站点名称",
"host": "IP地址",
"username": "登录名",
"password": "登录密码",
"protocol": "sftp",
"port": 22,
"remotePath": "链接路径",
"uploadOnSave": true,//上传下载服务
}
