kail-xshell-ftp
2020-06-22

1、是否打开ssh  服务?

2、打开后是否需要本地数据库支持？  kali 升级后 msf数据库默认是 postgresql  要打开服务

service ssh start

service postgresql start

解决以上问题 就能顺利连接xshell。

用Xshell root连接时显示ssh服务器拒绝了密码，应该是应该是sshd的设置不允许root用户用密码远程登录 
修改 /etc/ssh/sshd_config文件，注意，安装了openssh才会有这个文件，如果文件不存在请检查是否安装了openssh。

vim /etc/ssh/sshd_config
1
找到代码

Authentication:

LoginGraceTime 120
PermitRootLogin prohibit-password
StrictModes yes
1
2
3
4
5

改成

Authentication:

LoginGraceTime 120
PermitRootLogin yes
StrictModes yes
1
2
3
4
然后输入命令

/etc/init.d/ssh restart
1
重启ssh服务即可。
